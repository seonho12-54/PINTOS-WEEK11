# Legal and Ethical Issues (법적 및 윤리적 문제)

Pintos의 원본 버전은 자유로운 사용, 수정, 배포를 허용하는 관대한 license(사용 허가 조건)로 배포됩니다. Pintos 작업을 하는 학생과 다른 사람들은 자신이 작성한 code(소스 코드)를 소유하며, 어떤 목적으로든 사용할 수 있습니다. Pintos는 *MERCHANTABILITY*(상품성)나 *FITNESS FOR A PARTICULAR PURPOSE*(특정 목적 적합성)에 대한 보증을 포함하여 어떠한 *WARRANTY*(보증)도 제공하지 않습니다. license의 상세 내용과 보증 부재에 대해서는 License file을 참조하세요. 하지만 pintos-kaist의 수정된 code는 같은 license를 따르지 않습니다. 대신 *RECEX SHARED SOURCE LICENSE*를 사용하며, 이는 이 code를 기반으로 한 여러분의 작업물을 *재배포해서는 안 된다*는 뜻입니다. 자세한 내용은 `LICENSE`를 참조하세요.

KAIST CS330 course(수업)의 맥락에서는 온라인이나 다른 곳에서 제공되는 homework solution(과제 해답)을 읽지 않음으로써 honor code(명예 규정)의 정신과 문구를 모두 존중해 주세요. Linux나 FreeBSD 같은 다른 operating system kernel(운영체제 핵심부)의 source code를 읽는 것은 허용되지만, 그 code를 문자 그대로 복사해서는 안 됩니다.
