프로세스 종료 메시지

프로세스 종료 메시지(Process Termination Message)

프로세스(process) 종료 메시지를 출력하십시오.

사용자 프로세스(process)가 exit를 호출했거나 다른 어떤 이유로 종료될 때마다, 아래 printf 호출과 같은 형식으로 프로세스(process)의 이름과 종료 코드(exit code)를 출력하십시오.

printf ("%s: exit(%d)\n", ...);

출력되는 이름은 fork()에 전달된 전체 이름이어야 합니다. 사용자 프로세스(process)가 아닌 커널(kernel) 스레드(thread)가 종료될 때나 halt 시스템 콜(system call)이 호출될 때는 이 메시지를 출력하지 마십시오. 프로세스(process)가 로드에 실패한 경우에는 이 메시지를 출력해도 되고 출력하지 않아도 됩니다.

이 경우를 제외하고, 제공된 Pintos가 이미 출력하지 않는 다른 메시지는 출력하지 마십시오. 디버깅(debugging) 중에는 추가 메시지가 유용할 수 있지만, 채점 스크립트를 혼란스럽게 만들어 점수를 낮출 수 있습니다.
