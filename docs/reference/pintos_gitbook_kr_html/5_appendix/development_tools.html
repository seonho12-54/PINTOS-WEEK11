개발 도구

다음은 코드를 개발할 때 유용하게 쓸 수 있는 개발 도구(development tools)입니다.

태그(tags)

태그(tags)는 프로그램에 선언된 함수와 전역 변수에 대한 색인입니다. Emacs와 vi를 비롯한 많은 편집기가 이를 사용할 수 있습니다. Makefile은 pintos/에서 make TAGS 명령으로 Emacs 방식 태그를 만들고, make tags로 vi 방식 태그를 만듭니다.

Emacs에서는 현재 창에서 태그를 따라가려면 M-.을 사용하고, 새 창에서는 C-x 4 .을, 새 프레임에서는 C-x 5 .을 사용합니다. 이 명령들 중 어느 것을 실행할 때 커서가 심볼 이름 위에 있으면, 그 심볼이 기본 대상이 됩니다. 태그 이름에 여러 정의가 있으면 M-0 M-.으로 다음 정의로 이동합니다. 마지막으로 따라간 태그 이전 위치로 돌아가려면 M-*를 사용합니다.

cscope

cscope 프로그램도 프로그램에 선언된 함수와 변수에 대한 색인을 제공합니다. 태그 기능에는 없는 몇 가지 기능도 갖추고 있습니다. 특히 주어진 함수가 프로그램 안에서 호출되는 모든 위치를 찾을 수 있습니다.

Makefile은 pintos/에서 make cscope로 호출될 때 cscope 색인을 만듭니다. 색인이 생성되면 셸 명령줄에서 cscope를 실행합니다. 일반적으로 명령줄 인수는 필요하지 않습니다. 그런 다음 방향키로 터미널 아래쪽 근처에 나열된 검색 기준 중 하나를 고르고, 식별자를 입력한 뒤 Enter를 누릅니다. 그러면 cscope가 터미널 위쪽에 일치 항목을 표시합니다. 방향키로 특정 항목을 고를 수 있으며, 그 상태에서 Enter를 누르면 cscope가 기본 시스템 편집기1를 실행하고 커서를 해당 항목 위치에 놓습니다. 새 검색을 시작하려면 Tab을 입력합니다. cscope를 종료하려면 Ctrl-d를 입력합니다.

Emacs와 일부 vi 버전에는 cscope 전용 인터페이스가 있습니다. 이 인터페이스를 사용하는 방법은 cscope 홈페이지 (http://cscope.sourceforge.net)를 참조하십시오.

1. 일반적으로 vi입니다. vi를 종료하려면 :q Enter를 입력합니다. ↩
